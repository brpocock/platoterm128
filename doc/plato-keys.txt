Sh+Tab = CR can be entered also by typing Line Feed.


The RESTORE key will be used for terminal controls.

Shift Lock is mechanically identical to the left Shift key and does what
is expected.

CAPS Lock  is translated to a  Shifted version of only  the letter keys,
leaving the other keys unshifted.

The numeric  keypad, unshifted, enters digits  indistiguishably from the
number row on the typewriter section, but  the + and - keys are remapped
as  × and  ÷  instead. Enter  is indistinguishable  from  Return or  the
top-row → key (all  send NEXT). The shifted digits 1, 2, 3,  4, 6, 7, 8,
and   9  send   letters   to   mimic  the   arrow   placements  on   the
typewriter section.
